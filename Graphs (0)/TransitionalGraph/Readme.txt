Граф с переходами. 
Цель: предосталвение базы для машины состояний например как в анимационной машине состояний.
Отличительная особенность в том что последовательные переходы должны хранить результат работы если это требуется чтобы например в первой ноде посчитать результат1 который будет нужен в 4 ноде.
Каждый эдж это переход
Каждая нода это состояние (процесс)
